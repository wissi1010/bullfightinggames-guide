# bullfightinggames-guide

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库本身不针对任何特定域名或网站，仅作为各类静态页面资源的整理与托管空间。

## 目录说明

- `pages/`：存放所有独立 HTML 页面文件，每个页面为一个完整的可访问文档。
- `assets/`：存放页面引用的静态资源，如图片、样式表、脚本等。
- `docs/`：存放与页面相关的说明文档或附加资料。

## 页面归档说明

本仓库收录的页面均为独立编写，不依赖外部后端服务。页面内容可能涉及不同主题或用途，每个页面均保留原始结构与功能。归档目的为便于版本管理、历史查阅及静态部署。

## 维护说明

- 页面更新或新增时，请将 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。
- 如有资源文件变动，请同步更新 `assets/` 目录下对应文件。
- 如需对页面进行说明或备注，可在 `docs/` 中添加相应文档。
- 本仓库不接受包含恶意代码、违法内容或侵犯他人权益的页面提交。

## 贡献指南

欢迎提交改进建议或新增页面。提交前请确保：
- 页面内容符合开源社区规范。
- 所有资源文件均已包含在仓库中，不依赖外部链接。
- 提供简短的页面说明，便于他人理解用途。
